# Node.js Version
 
To change the Node.js version, access your project on bohr.io, go to Settings, Environment Variables, and add the variable BOHR_FUNCTION_RUNTIME with one of the values below:

- `Nodejs16.x`
- `Nodejs18.x`
- `Nodejs20.x`
